### 할인 행사

```python
def solution(want, number, discount):
    answer = 0
    
    for i in range(len(discount)-9):
        lst = discount[i:i+10]   # 회원인 기간동안 할인하는 제품
        cnt = 0
        for j in range(len(want)):
            if lst.count(want[j]) < number[j]:  # 원하는 수량 만큼 없다면
                break
            else:
                cnt += 1
                
        if cnt == len(want):
            answer += 1     
    
    return answer
```
