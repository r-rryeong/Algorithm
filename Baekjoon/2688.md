### 줄어들지 않아

```python
T = int(input())   # 테스트 케이스 개수
dp = [[0]*10 for _ in range(65)]
dp[1] = [1]*10     # 한자리 수는 0~9의 개수가 1이다
for _ in range(T):
    n = int(input())   # n자리 수
    for i in range(2, n+1):
        # i 자리 수의 0의 개수가 (i-1) 자리 수의 총 경우의 수
        dp[i][0] = sum(dp[i-1])
        for j in range(1, 10):
            dp[i][j] = dp[i][j-1] - dp[i-1][j-1]

    print(sum(dp[n]))
```
