### 모든 순열

```python
def solution(arr):

    if len(arr) == N:
        print(*arr)
        return

    for i in range(1, N+1):
        if i in arr:
            continue
        else:
            arr.append(i)
            solution(arr)
            arr.pop()


N = int(input())
arr = []
solution(arr)
```
