### 캠프가는 영식

```python
N, T = map(int, input().split())   # 버스의 개수, 터미널에 도착한 시간
answer = 1e9
for _ in range(N):
    # 버스 시작 시각, 간격, 대수
    s, l, c = map(int, input().split())
    for _ in range(c):
        if T <= s:
            answer = min(answer, s-T)
            break
        s += l   # 다음 버스 시간

print(answer if answer != 1e9 else -1)
```
