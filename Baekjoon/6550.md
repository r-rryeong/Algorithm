### 부분 문자열

```python
import sys
sys.stdin = open('input.txt', 'r')
input = sys.stdin.readline

while True:
    try:
        S, T = input().split()
        i = 0
        for t in T:
            if S[i] == t:
                i += 1
                if i >= len(S):
                    print("Yes")
                    break
        if i < len(S) - 1:
            print("No")

    except:
        break
```
