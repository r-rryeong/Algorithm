### 단어 정렬

```python
import sys
input = sys.stdin.readline

N = int(input())
lst = []
for _ in range(N):
    s = input().rstrip()
    # 이미 존재하는 단어는 추가하지 않는다
    if (len(s), s) not in lst:
        lst.append((len(s), s))

lst.sort()    # 단어의 길이 우선, 다음으로 알파벳순으로 오름차순 정렬
for x in lst:
    print(x[1])   # 단어 출력
```
