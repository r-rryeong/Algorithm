### 쇠막대기

```python
stick = list(input())
stack = []
answer = 0
for i in range(len(stick)):
    if stick[i]=='(':
        stack.append(stick[i])
        if stick[i+1]==')':
            stack.pop()
            answer += len(stack)
    else:
        if stick[i-1]=='(':
            continue
        if not stack:
            continue
        stack.pop()
        answer += 1

print(answer)
```
